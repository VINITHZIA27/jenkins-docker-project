📌 Project Overview

This project demonstrates a complete CI/CD pipeline where application code is automatically built, containerized, and deployed using modern DevOps tools.

🛠 Tools & Technologies Used

  • Git & GitHub – Source Code Management
  • Jenkins – Continuous Integration & Continuous Deployment
  • Docker – Containerization
  • AWS EC2 – Cloud Infrastructure
  • Nginx – Web Server

⚙ CI/CD Workflow

  • Developer pushes code to GitHub
  • Jenkins automatically pulls the code
  • Docker image is built
  • Container is deployed on EC2
  • Application becomes live
